# 🧮 Menu Driven Calculator (JavaScript)

A simple and interactive **Menu-Driven Calculator** built using **Vanilla JavaScript**, integrated inside a popup section on the homepage.

This project demonstrates fundamental JavaScript concepts such as DOM manipulation, conditional logic, and event handling in a clean and structured way.

---

## 🚀 Features

- 📌 Menu-based operation selection  
- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🪟 Popup modal interface  
- ⚡ Instant result display  
- 💻 Beginner-friendly implementation  

---

## 🛠️ Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla JS)** – Logic and functionality

CALCULATOR_JS/

---

## 🧠 How It Works

1. The user opens the homepage.
2. A popup calculator is triggered.
3. The user selects an operation from the menu.
4. Two numbers are entered.
5. JavaScript processes the selected operation using conditional logic.
6. The result is displayed instantly inside the popup.

The calculator uses structured conditional statements (`if-else` or `switch`) to determine the selected arithmetic operation.
